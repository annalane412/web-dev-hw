Important Note!

Keeping the two individual cords seperate is extremely important.
Most indoor cords will have writing on one, but not the other.
If this is NOT the case, you will need to mark one of the cords on both sides of the break.
I suggest using a permanent marker.
This will ensure that you are able to reconnect each wire to the same one it was originally connected to.

Materials

  1. wirecutters
  2. scissors - not shown
  3. rounded pliars (can use tweezers)
  4. electrical tape (preferably the same color as the cord)
  5. a steady hand

First, let's cut the wires in half.

Make sure the wires are even.
Use scissors to score plastic wire coating.
Gently pull off approximately 1 in. of wire coating.

Repeat for all four wire ends.

Next, twist wires together.
Remember, this is when it is essential to connect them correctly.
I suggest using the soft pliars and a twisting motion.

Use two small squares of electrical tape to seperate wires on the inside.

Make a tape loop.

Insert loop of tape between wires.

Cut tops and bottoms of tape loop.

Wrap sides to completely seperate and cover each individual wire.

Wrap each wire thouroughly and seperately.

Wrap tape around both wires.

And VOILA!
